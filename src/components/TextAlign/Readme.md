TextAlign standard example:

```js
<TextAlign>
  Hello World
</TextAlign>
```

TextAlign centered example:

```js
<TextAlign align="center">
  Hello World
</TextAlign>
```


TextAlign right example:

```js
<TextAlign align="right">
  Hello World
</TextAlign>
```


TextAlign justified example:

```js
<TextAlign align="justified">
  Hello World
</TextAlign>
```
