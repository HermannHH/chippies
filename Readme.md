# BIT instructions

## Setup on Mac

1.  npm install bit-bin --global
2.  bit login

## Push component updates

1.  bit add src/components/\* <If component is not tracked yet>
2.  bit status
3.  bit tag --all [optional version]
4.  bit export hermannhh.chippies
